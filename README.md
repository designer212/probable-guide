# JayTech - Smart Tech Solutions Website

A professional, fully responsive mobile-first website tailored for JayTech, offering professional AI training, phone repairs, computer troubleshooting, networking, software installation, and creative graphic design services.

---

## 🚀 Features

* **Modern & Responsive Design:** Optimized primarily for mobile devices with clean dark/light UI accents.
* **Horizontal Sliding Stats:** Interactive mobile-friendly stats container with smooth `scroll-snap` behavior to display key business milestones without cluttering the screen.
* **Expandable Services Grid:** Interactive service cards that expand to reveal detailed service lists.
* **Floating WhatsApp Action Button:** Quick-access direct contact button for immediate customer inquiries.
* **Dynamic Navigation:** Fixed header with blur backdrop effect and responsive hamburger menu navigation.

---

## 🛠️ Built With

* **HTML5**
* **CSS3** (Custom Properties, Flexbox, Grid, Media Queries)
* **JavaScript** (For interactive elements like service toggles)
* **Google Fonts** (Poppins & Inter)
